Ҷанг барои чоп кардани пул аст.
Вазорати молия вомбаргҳои ҷангӣ мебарорад ва ба мардум эълон мекунад, ки онҳоро харанд. Агар касе нахарад, бонки марказӣ онҳоро мехарад. Ин аст оғози чоп кардани пул.
Пул ба занҷири таъминоти саноати ҷангӣ дар ширкатҳои гуногуни худи Амрико паҳн мешавад, аз ҷумла пуле, ки метавонад ба ҳама кишварҳои ҷаҳон дар замони ҷанг фурӯхта шавад, ки ин маблағи бузурги пул аст.
Пас аз он ки пули ширкати силоҳ ба системаи бонкдории тиҷоратӣ ворид мешавад, ҳаҷми пул дар система барои Иёлоти Муттаҳида 10-20 маротиба зиёд мешавад (дар ҳоле ки Таиланд тақрибан 6-7 маротиба аст) тавассути додани қарзҳо такроран дар ҳамон пул (чоп кардани пул ё зиёд кардани ҳаҷми пул дар иқтисодиёт, ки дар сатҳи бонкдории тиҷоратӣ Fractional Reserve Banking рух медиҳад) ба системаи иқтисодии Амрико ҷорӣ мешавад, ки нуқтаи ибтидоии крани падидаи Кантилон Эффект аз системаи молиявии ҷаҳон аз дастгоҳи асосии чопии ҷаҳон, яъне Иёлоти Муттаҳида мебошад.
Ин ҳавасмандгардонии иқтисодӣ тавассути баҳонаи ҷанг бо саноати ҷангӣ аст. Ҳар як чоп кардани пул аз ҷониби дастгоҳи чопӣ дуздии қудрати харидорӣ ё дуздии арзиши пул аз ҳама дар саросари ҷаҳон аст. Онҳо ҳама кишварҳоро ғорат мекунанд. Чӣ қадаре ки кишваре захираҳои асъории доллари баланд дошта бошад, ҳамон қадар бештар ғорат кардан мумкин аст. Ба онҳо фарқ надорад, ки дар ҷанг мағлуб мешаванд ё пирӯз. Онҳо танҳо ба он аҳамият медиҳанд, ки чӣ қадар пулро метавонанд чоп кунанд ва тамоми ҷаҳонро ғорат кунанд. (Вақте ки бӯҳрони иқтисодӣ ба амал меояд, ширкатҳои саросари ҷаҳон хароҷотро бо ихтисори коргарон ва иваз кардани онҳо бо AI-и арзонтар коҳиш медиҳанд, аммо ҳоло ҳама дар саросари ҷаҳон бояд қудрати коркарди AI-ро аз Амрико харанд.)

П.С. Амрико ҳеҷ гоҳ ҷанг накардааст, то дар ҷанг пирӯз шавад. Ғалаба дар ҷанг ҳадафи асосии онҳо нест, балки онҳо барои чоп кардани пул ҷанг мекунанд (эҷоди таваррум барои коҳиш додани арзиши қарзҳо). Ҷанг як сабаби хуб ва баҳонаи хуб барои чоп кардани пул аст. Вақте ки пул чоп кардан мумкин аст, онҳо метавонанд пулро аз тамоми ҷаҳон дузданд, на танҳо аз рақибони ҷанг. Аммо чоп кардани пул дар давоми ҷанг ба онҳо имкон медиҳад, ки ҳама кишварҳоро ғорат кунанд, бахусус кишварҳое, ки захираҳои зиёд доранд. Чӣ қадаре ки захираҳо дар доллар зиёд бошанд, ҳамон қадар бештар ғорат кардан мумкин аст (ҳатто агар касе дар оянда ба қудрати бузург табдил ёбад, онҳо тақрибан ҳамон одатҳоро истифода мебаранд. Аксарияти таърихи ҷаҳон дар гузашта фарқ надорад).
Ҳамаи инро бо фалсафаи молиявии Hard Money-и муосир, яъне Bitcoin ҳал кардан мумкин аст.
Тилло низ дар замони сулҳ Hard Money-и хуб аст, аммо таърих ба мо мегӯяд, ки тилло дар замони ҷанг нисбат ба дигар вақтҳо осонтар ғорат мешавад, бахусус ҳилаҳои ҳукуматҳои гуногун дар гузашта, ки аксар вақт тиллои мардуми худро барои истифода ҳамчун захира дар ҷанг мусодира мекарданд, вақте ки онҳо аз ҷиҳати молиявӣ ва молиявӣ дар ҳолати вазнин қарор доштанд.
