Iimphi zokugadangisa imali
Iminyango yezeMali ikhupha amabhondi wempi, imemezela ukuthi abantu abathenge. Nange kungekho obathengako, ibhanga elikhulu lizozithengela. Lokhu kuqala kwegadangiso lemali.
Imali izokusabalala kuyo yoke i-supply chain yezimboni zempi, emakampanini ahlukahlukeneko we-America, kanye nemali engathengisa izikhali kiyo yoke imibuso yephasi ngesikhathi sempi, okuyimali enengi khulu godu.
Ngemva kwalokho, nange imali yamakampani wezikhali ingena ehlelweni lamabhanga wezentengiselwano, inani lemali ehlelweni lizokukhuphuka nge-10-20 ngokuphindaphindeka kwe-United States (eThailand, cishe i-6-7 ngokuphindaphindeka). Ngokukhupha imalimboleko ephindaphindwako ngemali efanako (ukugadangisa imali namkha ukukhuphula inani lemali ehlelweni lezomnotho elenzeka ezingeni lamabhanga wezentengiselwano, i-Fractional Reserve Banking) igelezelela ehlelweni lezomnotho we-America njengomthombo wokuqala we-Cantillon effect wehlelo lezezimali lephasi ukusuka emshinini wokugadangisa oyinhloko wephasi, okuyi-United States.
Lokhu kukhuphula umnotho ngesizathu sempi ngeemboni zempi. Ukugadangisa imali ngakunye komshini wokugadangisa imali kukubutjha amandla wokuthenga namkha ukubutjha inani lemali kubo boke abantu ephasini. Kubutjha yoke imibuso. Nange umbuso unomthamo omkhulu wemali yamaDola, kulapho ungabutjhwa khona khulu. Abanandaba nokuthi bayahlula namkha bayahlulwa empini. Bakhathazeka kuphela ngokuthi bangagadangisa imali engangani ukubutjha abantu boke ephasini. (Enye into, nange kuvela ubunzima bezomnotho, amakampani ephasini loke azokunciphisa iindleko ngokuxotjha abasebenzi begodu abathathele indawo nge-AI etjhiphileko, kodwana woke umuntu ephasini loke kufuze athenge amandla wokucubungula we-AI e-America njenganje.)
