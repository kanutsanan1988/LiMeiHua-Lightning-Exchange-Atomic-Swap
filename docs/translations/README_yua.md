Jump'éel ba'ate'el utia'al u imprimir ta'ak'in
Le Secretaría ti' Hacienda ku ts'aik bonos ti' ba'ate'el, ku ya'alik ti' le kaajo' u maniko'ob. Wa mix máak ku manik, le Banco Central yaan u manik. Lela' leti' u káajal le impresión ti' ta'ak'ino'.
Le ta'ak'ino' yaan u t'i'ikbal ti' le cadena suministro ti' le industria ti' ba'ate'elo', ti' jejeláas empresas ti' Estados Unidos, bey xan le ta'ak'in ku páajtal u ko'onol armas ti' tuláakal le lu'umo'obo' ti' k'iin k'aasil, jump'éel nuxi' cantidad ta'ak'in xan.
Ts'o'okole', le kéen okok le ta'ak'in ti' le empresa ti' armas ti' le yaan bancario comercial, le cantidad ti' ta'ak'in ti' le yaan yaan u ya'abtal 10-20 u téenel utia'al Estados Unidos (ti' Tailandia yaan kex 6-7 u téenel). Tu yo'olal u ts'a'abal p'aax ya'ab u téenel yéetel le k'éexel ta'ak'ino' (le impresión ti' ta'ak'in wa le ya'abtal ti' le cantidad ti' ta'ak'in ti' le yaan económico ku yúuchul ti' u nivel le bancos comerciales Fractional Reserve Banking) ku t'i'ikbal ti' le yaan económico ti' Estados Unidos bey u káajal le fenómeno Cantillon Effect ti' le yaan financiero mundial ti' u noj beeta'al le yóok'ol kaabo', Estados Unidos.
Lela' jump'éel estimulación económica ti' u yo'olal jump'éel ba'ate'el yéetel le industria ti' ba'ate'elo'. Le impresión ti' ta'ak'in ti' cada beeta'al ta'ak'in leti' u robamiento ti' le páajtalil u maan wa le robamiento ti' u valor le ta'ak'in ti' tuláakal le máako'obo' ti' yóok'ol kaab. Robar ti' tuláakal le lu'umo'obo'. Cuanto maas yaan ti' jump'éel lu'um reservas ti' dólares, maas je'el u páajtal u robarta'al. Ma' u preocupar u wa ku ganaro'ob wa ku p'áatalo'ob ti' le ba'ate'elo'. U preocupar u jayp'éel ta'ak'in je'el u páajtal u imprimir utia'al u robarto'ob ti' tuláakal le máako'obo' ti' yóok'ol kaab. (Uláak' ba'al, le kéen yanak jump'éel crisis económica, le jejeláas empresas ti' yóok'ol kaab yaan u xu'ulsiko'ob le gastos, u tseliko'ob le meyajo'obo' yéetel u sutk'eso'ob yéetel AI maas barato, ba'ale' tuláakal le máako'obo' ti' yóok'ol kaab k'a'abéet u maniko'ob le páajtalil u procesamiento ti' AI ti' Estados Unidos beoraa').

Le kéen páajchajak u imprimir ta'ak'ine', je'el u páajtal u robarto'ob ta'ak'in ti' tuláakal le máako'obo' ti' yóok'ol kaab. Ma' chéen u robarto'ob ti' le opositores ti' jump'éel ba'ate'el, ba'ale' le impresión ti' ta'ak'in ichil jump'éel ba'ate'elo' je'el u páajtal u robarto'ob ti' tuláakal le lu'umo'obo', especialmente le lu'umo'ob yaan ya'ab reservas ti'obo'. (Kex wa uláak' máak ku na'akal bey jump'éel superpotencia, yaan u meyajtik le k'ek'el ku beetik lela' tuláakal, ma' jach yaanal, le ya'abkach ti' le historia mundial máaniko' ma' yaanal).
