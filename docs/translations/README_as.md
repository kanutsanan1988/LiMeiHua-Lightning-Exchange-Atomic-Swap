যুদ্ধৰ উদ্যোগৰ জৰিয়তে যুদ্ধৰ অজুহাতত এইটো অৰ্থনীতিৰ উদ্দীপনা। টকা ছপা কৰাৰ প্ৰতিটো কাৰ্য্য হৈছে বিশ্বৰ সকলোৰে পৰা ক্ৰয় ক্ষমতা বা টকাৰ মূল্য চুৰি কৰা। সকলো দেশক চুৰি কৰা হয়। যিমান বেছি দেশৰ ডলাৰৰ ৰিজাৰ্ভ থাকে, সিমান বেছি চুৰি কৰিব পাৰি। তেওঁলোকে যুদ্ধত হাৰিলে নে জিকিলে সেইটো গুৰুত্ব নিদিয়ে, তেওঁলোকে কেৱল গুৰুত্ব দিয়ে যে তেওঁলোকে গোটেই বিশ্বৰ মানুহৰ পৰা কিমান টকা ছপা কৰি চুৰি কৰিব পাৰে। (আৰু এটা কথা, যেতিয়া অৰ্থনৈতিক সংকট হয়, বিশ্বৰ কোম্পানীবোৰে কৰ্মচাৰীক ছাঁটাই কৰি খৰচ কম কৰে আৰু তেওঁলোকৰ ঠাইত কম খৰচী এআই ব্যৱহাৰ কৰে, কিন্তু এতিয়া গোটেই বিশ্বই আমেৰিকাৰ পৰা এআই প্ৰচেছিং ক্ষমতা কিনিব লাগিব)।
