Պատերազմ փող տպելու համար։
Ֆինանսների նախարարությունը թողարկում է պատերազմական պարտատոմսեր և հայտարարում, որ հանրությունը պետք է գնի դրանք։ Եթե ոչ ոք չգնի, ապա կենտրոնական բանկը կգնի։ Ահա թե ինչպես է սկսվում փողի տպագրությունը։
Փողը կտարածվի պատերազմական արդյունաբերության մատակարարման շղթայում, ամերիկյան տարբեր ընկերություններում, ինչպես նաև գումար, որը կարող է զենք վաճառել աշխարհի բոլոր երկրներին պատերազմի ժամանակ, ինչը նույնպես հսկայական գումար է։
Այնուհետև, երբ զենք արտադրող ընկերությունների փողը մտնում է առևտրային բանկային համակարգ, համակարգում փողի քանակը կավելանա 10-20 անգամ ԱՄՆ-ի համար (իսկ Թաիլանդում մոտ 6-7 անգամ), վարկավորման բազմաթիվ ցիկլերի միջոցով նույն գումարով (փողի տպագրություն կամ փողի զանգվածի ավելացում տնտեսական համակարգում, որը տեղի է ունենում առևտրային բանկային մակարդակում՝ Fractional Reserve Banking), որը հոսում է դեպի ամերիկյան տնտեսական համակարգ, որպես համաշխարհային ֆինանսական համակարգի Cantillon Effect-ի սկզբնական կետ, որի հիմնական տպագրական մեքենան ԱՄՆ-ն է։
Սա տնտեսության խթանում է պատերազմի պատրվակով՝ պատերազմական արդյունաբերության միջոցով։ Փողի տպագրական մեքենայի կողմից փողի յուրաքանչյուր տպագրություն գնողունակության գողություն է կամ փողի արժեքի գողություն աշխարհի բոլոր մարդկանցից։ Գողանում են բոլոր երկրները։ Որքան շատ դոլարային պահուստներ ունի մի երկիր, այնքան ավելի շատ կարող են գողանալ։ Նրանց չի հետաքրքրում՝ կհաղթեն պատերազմում, թե ոչ։ Նրանց միայն հետաքրքրում է, թե որքան փող կարող են տպել և գողանալ ամբողջ աշխարհից։ (Մի կողմից, երբ տնտեսական ճգնաժամ է տեղի ունենում, աշխարհի տարբեր ընկերություններ կրճատում են ծախսերը՝ աշխատանքից ազատելով աշխատողներին և փոխարինելով նրանց ավելի էժան արհեստական բանականությամբ, բայց աշխարհի բոլոր մարդիկ այժմ պետք է գնեն արհեստական բանականության հաշվողական հզորությունը Ամերիկայից)։
