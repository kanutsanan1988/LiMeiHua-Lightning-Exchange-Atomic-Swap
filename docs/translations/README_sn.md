Hondo yekudhinda mari
Bazi rezvemari rinoburitsa mabhondi ehondo, richizivisa vanhu kuti vatenge. Kana pasina anotenga, bhanga guru richazvitenga. Uku ndiko kutanga kwekudhinda mari.
Mari ichapararira kuenda kumaketani ekugovera indasitiri yehondo mumakambani akasiyana-siyana eAmerica, kusanganisira mari inogona kutengesa zvombo kunyika dzese pasi rose munguva yehondo, iyo iri mari yakawanda zvakare.
Ipapo kana mari yekambani yezvombo yapinda mugadziriro yemabhanga ekutengeserana, huwandu hwemari mugadziriro huchawedzera ne10-20 nguva kuUnited States (Thailand iri pa6-7 nguva). Uku kukura kwehupfumi kuburikidza nekupokana kwehondo neindasitiri yehondo.
