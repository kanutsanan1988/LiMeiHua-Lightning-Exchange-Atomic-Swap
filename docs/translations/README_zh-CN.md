为印钞而战

钱就会流向战争工业的供应链，流向美国各公司。还有一笔钱，可以在战时出售武器给世界各国，这笔钱也是一个巨大的数目。
然后，当武器公司的钱进入商业银行系统时，系统中的货币量将增加 10-20 倍（对于美国而言）（泰国约为 6-7 倍），通过在同一笔钱上多次循环贷款（在商业银行层面发生的印钞或增加经济体系中的货币量，即部分准备金银行制度），流入美国经济，成为世界金融体系坎蒂隆效应的源头，而世界主要的印钞机就是美国。
这是通过战争的借口，以战争工业来刺激经济。每次印钞机的印钞，都是在掠夺购买力，或者说是掠夺全世界所有人的货币价值。掠夺所有国家。哪个国家的美元储备越高，就越能掠夺得越多。他们不在乎战争是输是赢，他们只在乎他们能印多少钱来掠夺全世界的人。（另外，一旦发生经济危机，世界各地的公司都会通过解雇工人来降低成本，然后用更便宜的 AI 来代替，但现在全世界的人都必须购买美国的 AI 计算能力）。
