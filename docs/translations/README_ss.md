Imphi yekucala imali.
Litiko Letetimali likhipha tibopho temphi, limemetela kutsi bantfu batitsenge. Nangabe kute lotsengako, libhange lelikhulu litawutitsenga. Lena yindlela kucala ngayo kucala imali.
Imali itawusakateka kuyo yonkhe i-supply chain yemboni yetempi, etinkampanini letihlukahlukene eMelika, kanye nemali yekutsengisa tibhamu kuwo onkhe emave emhlabeni ngesikhatsi semphi, nayo leyo yimali lenkhulu.
Ngemuva kwaloko, uma imali yenkampani yetibhamu ingena kuluhlelo lwebhange lekuhweba, linani lemali kuluhlelo liyakwenyuka nge-10-20 kuMelika (eThailand cishe yi-6-7). Ngekutsi kuniketwe imali ngekuyiphindzaphindza tikhatsi letinyenti ngemali lefanako (kucala imali nobe kwandzisa linani lemali kuluhlelo lwetemnotfo lolwenteka ezingeni lebhange lekuhweba i-Fractional Reserve Banking) igeleta kuluhlelo lwetemnotfo lwaseMelika njengemthombo we-Cantillon Effect yeluhlelo lwetimali lwemhlaba kusuka kumshini wekucala lomkhulu emhlabeni, iMelika.
Loku kukhutsata temnotfo ngekusebentisa imboni yetempi njengesizatfu semphi. Kucala imali ngakunye kumshini wekucala imali kufana nekweba emandla ekutsenga nobe kweba lizinga lemali kubo bonkhe bantfu emhlabeni jikelele. Yeba kuwo onkhe emave. Nakulapho live linemali lenyenti yema-dollar, kulapho lingeba khona kakhulu. Abanandzaba nekutsi bayayinqoba yini imphi nobe cha. Banendzaba nekutsi bangacala imali engakanani yekweba kubantfu emhlabeni wonkhe. (Ngalenye indlela, uma kuba nenkinga yetemnotfo, tinkampani emhlabeni wonkhe titawunciphisa tindleko ngekutsi ticotshe tisebenti bese titsi tisebentise i-AI leshibhile, kodvwa wonkhe umuntfu emhlabeni wonkhe kufanele athenge emandla ekucubungula e-AI eMelika nyalo.)

P.S. IMelika ayikaze iyilwe imphi ngobe ifuna kuyinqoba. Kunqoba imphi akusiyo injongo yayo lenkhulu, kodvwa ilwa imphi yekucala imali (yekwakha i-inflation yekunciphisa lizinga letikweleti). Imphi sisizatfu lesihle sekucala imali. Uma ungacala imali, ungayeba kubantfu emhlabeni wonkhe, hhayi nje kuletimbangi emphini, kodvwa ngekucala imali ngesikhatsi semphi, ungayeba kuwo onkhe emave, ikakhulukati emave lanemali lenyenti. Uma imali yakho iyinyenti ngema-dollar, kulapho ungayeba khona kakhulu. (Nobe ngabe ngubani lotawuba ngumbusi, utawusebentisa luhlobo lolufanako lwesimilo, cishe aluhlukile. Kunyenti emlandvweni wemhlaba lokungahlukile.)
Konkhe loku kungalungiswa nge filosofi yetimali ye-Hard Money yalamuhla, i-Bitcoin.
Igolide nayo iyi-Hard Money lenhle ngesikhatsi sekuthula, kodvwa umlando usitjela kutsi igolide livame kwebiwa melula ngesikhatsi semphi, ikakhulukati ngemacebo ahulumende lamanyenti esikhatsi lesendlulile lavame kuyala kutsi igolide lebantfu litsetfwe njengemcebo wekulwa imphi uma bafikelwa yinkinga yetimali.
Lamuhla, lithulusi lekuthula selikhona, kodvwa alikasondeleki futsi alikasabalali kubantfu labanyenti. Ngako-ke, imphi lenkhulu (esiprofethweni setinkholo letinyenti) letako itawuba yimphi yekugcina kule-bhadrakappa. Ngemuva kwaloko, kutawungena inkathi yekuthula kwaphakade (kwesikhashana) kuze kube sekupheleni kwale-bhadrakappa.
