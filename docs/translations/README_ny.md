Kupambana nkhondo si cholinga chawo chachikulu, koma amamenya nkhondo kuti asindikize ndalama (kupanga inflation kuti achepetse mtengo wa ngongole). Nkhondo ndi chifukwa chabwino, chifukwa chabwino chosindikizira ndalama. Akasindikiza ndalama, amatha kubera ndalama kwa anthu padziko lonse lapansi, osati kungobera omwe akutsutsana nawo pankhondo, koma kusindikiza ndalama panthawi yankhondo, amatha kubera mayiko onse, makamaka mayiko omwe ali ndi ndalama zambiri.
