# ᐊᓂᔑᓈᐯᒧᐎᓐ (Ojibwe)

ᐊᐦᑑᑫᐎᓂ ᑰᔅᑫᐎᐣ ᑫᐦᒋ ᑫᐦᒋ ᐲᒫᑎᓯᐎᐣ ᑫᑫᐎᓐᑕᒣᐠ
