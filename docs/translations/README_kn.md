# ಕನ್ನಡ (Kannada)

ಸೋರ್ಸ್ ಕೋಡ್ ಯುದ್ಧವನ್ನು ನಿಲ್ಲಿಸುತ್ತದೆ
ಹಾರ್ಡ್ ಮನಿಯೊಂದಿಗೆ ಹಣಕಾಸು ತತ್ವಶಾಸ್ತ್ರವನ್ನು ಪ್ರವೇಶಿಸಿ
ಇದು ದೊಡ್ಡ ಯುದ್ಧವನ್ನು ನಷ್ಟಕ್ಕೆ ಮಾತ್ರ ಕಾರಣವಾಗುತ್ತದೆ.
ಇನ್ನು ಮುಂದೆ ಯುದ್ಧದಿಂದ ಯಾರೂ ಲಾಭ ಪಡೆಯಲು ಸಾಧ್ಯವಿಲ್ಲ.
ಏಕೆಂದರೆ ಯುದ್ಧವೆಂದರೆ ವ್ಯಾಪಾರ ಮತ್ತು ವ್ಯಾಪಾರವೆಂದರೆ ಯುದ್ಧ. ಇತಿಹಾಸವು ನಮಗೆ ಹಾಗೆ ಹೇಳುತ್ತದೆ. ಲಾಭ ಇಲ್ಲದಿದ್ದಾಗ, ಯುದ್ಧ ಮಾಡಲು ಯಾರಿಗೂ ಪ್ರೇರಣೆ ಇರುವುದಿಲ್ಲ.

ಹಣವನ್ನು ಮುದ್ರಿಸಲು ಯುದ್ಧ.
ಹಣಕಾಸು ಸಚಿವಾಲಯವು ಯುದ್ಧ ಬಾಂಡ್‌ಗಳನ್ನು ನೀಡುತ್ತದೆ ಮತ್ತು ಖರೀದಿಸಲು ಸಾರ್ವಜನಿಕರಿಗೆ ಘೋಷಿಸುತ್ತದೆ. ಯಾರೂ ಖರೀದಿಸದಿದ್ದರೆ, ಕೇಂದ್ರ ಬ್ಯಾಂಕ್ ಅದನ್ನು ಖರೀದಿಸುತ್ತದೆ. ಇದು ಹಣವನ್ನು ಮುದ್ರಿಸುವ ಆರಂಭವಾಗಿದೆ.
ಹಣವು ಯುದ್ಧ ಉದ್ಯಮದ ಪೂರೈಕೆ ಸರಪಳಿಗೆ ಹರಿಯುತ್ತದೆ, ಅಮೆರಿಕದ ವಿವಿಧ ಕಂಪನಿಗಳಿಗೆ, ಮತ್ತು ಯುದ್ಧದ ಸಮಯದಲ್ಲಿ ಪ್ರಪಂಚದಾದ್ಯಂತದ ಎಲ್ಲಾ ದೇಶಗಳಿಗೆ ಶಸ್ತ್ರಾಸ್ತ್ರಗಳನ್ನು ಮಾರಾಟ ಮಾಡಲು ಸಾಧ್ಯವಾಗುವ ಹಣದ ರಾಶಿಯೂ ಇದೆ, ಇದು ದೊಡ್ಡ ಮೊತ್ತದ ಹಣವೂ ಆಗಿದೆ.
ನಂತರ ಶಸ್ತ್ರಾಸ್ತ್ರ ಕಂಪನಿಗಳ ಹಣವು ವಾಣಿಜ್ಯ ಬ್ಯಾಂಕಿಂಗ್ ವ್ಯವಸ್ಥೆಯನ್ನು ಪ್ರವೇಶಿಸಿದಾಗ, ವ್ಯವಸ್ಥೆಯಲ್ಲಿನ ಹಣದ ಪ್ರಮಾಣವು ಯುನೈಟೆಡ್ ಸ್ಟೇಟ್ಸ್‌ಗೆ 10-20 ಪಟ್ಟು ಹೆಚ್ಚಾಗುತ್ತದೆ (ಥೈಲ್ಯಾಂಡ್‌ನಲ್ಲಿ ಇದು ಸುಮಾರು 6-7 ಪಟ್ಟು). ಮೂಲ ಹಣದಲ್ಲಿಯೇ ಹಲವಾರು ಬಾರಿ ಸಾಲವನ್ನು ಮರುಹಂಚಿಕೆ ಮಾಡುವ ಮೂಲಕ (ಹಣವನ್ನು ಮುದ್ರಿಸುವುದು ಅಥವಾ ವಾಣಿಜ್ಯ ಬ್ಯಾಂಕುಗಳ ಮಟ್ಟದಲ್ಲಿ ಆರ್ಥಿಕ ವ್ಯವಸ್ಥೆಯಲ್ಲಿ ಹಣದ ಪ್ರಮಾಣವನ್ನು ಹೆಚ್ಚಿಸುವುದು ಫ್ರಾಕ್ಷನಲ್ ರಿಸರ್ವ್ ಬ್ಯಾಂಕಿಂಗ್), ಅಮೆರಿಕದ ಆರ್ಥಿಕ ವ್ಯವಸ್ಥೆಗೆ ಹರಿಯುತ್ತದೆ, ಇದು ಪ್ರಪಂಚದ ಮುಖ್ಯ ಮುದ್ರಣಾಲಯವಾದ ಯುನೈಟೆಡ್ ಸ್ಟೇಟ್ಸ್‌ನಿಂದ ಜಾಗತಿಕ ಹಣಕಾಸು ವ್ಯವಸ್ಥೆಯ ಕ್ಯಾಂಟಿಲ್ಲನ್ ಪರಿಣಾಮದ ಆರಂಭಿಕ ಹಂತವಾಗಿದೆ.
ಇದು ಯುದ್ಧ ಉದ್ಯಮದ ಮೂಲಕ ಯುದ್ಧದ ನೆಪದಲ್ಲಿ ಆರ್ಥಿಕತೆಯನ್ನು ಉತ್ತೇಜಿಸುವುದು. ಪ್ರತಿ ಬಾರಿ ಹಣವನ್ನು ಮುದ್ರಿಸಿದಾಗ, ಅದು ಪ್ರಪಂಚದಾದ್ಯಂತದ ಎಲ್ಲರಿಂದಲೂ ಕೊಳ್ಳುವ ಶಕ್ತಿಯನ್ನು ಕದಿಯುವುದು ಅಥವಾ ಹಣದ ಮೌಲ್ಯವನ್ನು ಕದಿಯುವುದು. ಎಲ್ಲಾ ದೇಶಗಳನ್ನು ಕದಿಯಿರಿ. ಯಾವ ದೇಶವು ಹೆಚ್ಚಿನ ಡಾಲರ್ ಮೀಸಲು ಹೊಂದಿದೆಯೋ, ಅದನ್ನು ಹೆಚ್ಚು ಕದಿಯಬಹುದು. ಯುದ್ಧದಲ್ಲಿ ಗೆಲ್ಲುತ್ತಾರೋ ಅಥವಾ ಸೋಲುತ್ತಾರೋ ಎಂಬುದರ ಬಗ್ಗೆ ಅವರಿಗೆ ಕಾಳಜಿಯಿಲ್ಲ. ಅವರು ಪ್ರಪಂಚದಾದ್ಯಂತದ ಜನರನ್ನು ಎಷ್ಟು ಹಣವನ್ನು ಮುದ್ರಿಸಿ ಕದಿಯಬಹುದು ಎಂಬುದರ ಬಗ್ಗೆ ಮಾತ್ರ ಕಾಳಜಿ ವಹಿಸುತ್ತಾರೆ.
