യുദ്ധത്തിൽ നിന്ന് ആർക്കും ലാഭം നേടാൻ കഴിയില്ല.
കാരണം യുദ്ധം കച്ചവടമാണ്, കച്ചവടം യുദ്ധമാണ്. ചരിത്രം നമ്മെ പഠിപ്പിക്കുന്നത് അതാണ്. ലാഭമില്ലെങ്കിൽ ആർക്കും യുദ്ധം ചെയ്യാൻ തോന്നില്ല.

പണം അച്ചടിക്കുന്നതിനുള്ള യുദ്ധം
ധനകാര്യ മന്ത്രാലയം യുദ്ധ ബോണ്ടുകൾ പുറത്തിറക്കുന്നു, അത് വാങ്ങാൻ പൊതുജനങ്ങളോട് ആവശ്യപ്പെടുന്നു. ആരും വാങ്ങിയില്ലെങ്കിൽ സെൻട്രൽ ബാങ്ക് അത് വാങ്ങും. ഇങ്ങനെയാണ് പണം അച്ചടിക്കുന്നത് ആരംഭിക്കുന്നത്.
പണം യുദ്ധ വ്യവസായത്തിന്റെ സപ്ലൈ ചെയിനിലേക്ക് വ്യാപിക്കുന്നു. അമേരിക്കയിലെ വിവിധ കമ്പനികളിലേക്കും, യുദ്ധസമയത്ത് ലോകമെമ്പാടുമുള്ള എല്ലാ രാജ്യങ്ങൾക്കും ആയുധങ്ങൾ വിൽക്കാൻ കഴിയുന്ന പണവും ഇതിൽ ഉൾപ്പെടുന്നു, ഈ തുകയും വളരെ വലുതാണ്.
തുടർന്ന്, ആയുധ കമ്പനികളുടെ പണം വാണിജ്യ ബാങ്കിംഗ് സംവിധാനത്തിലേക്ക് പ്രവേശിക്കുമ്പോൾ, യുണൈറ്റഡ് സ്റ്റേറ്റ്സിനുള്ളിലെ പണത്തിന്റെ അളവ് 10-20 മടങ്ങ് വർദ്ധിക്കുന്നു (തായ്‌ലൻഡിൽ ഇത് ഏകദേശം 6-7 മടങ്ങാണ്). ഒരേ പണം തന്നെ പലതവണ വായ്പ നൽകി (പണം അച്ചടിക്കുക അല്ലെങ്കിൽ വാണിജ്യ ബാങ്കുകളുടെ തലത്തിൽ സാമ്പത്തിക സംവിധാനത്തിൽ പണത്തിന്റെ അളവ് വർദ്ധിപ്പിക്കുക, ഫ്രാക്ഷണൽ റിസർവ് ബാങ്കിംഗ്) അമേരിക്കൻ സമ്പദ്‌വ്യവസ്ഥയിലേക്ക് ഒഴുകുന്നു, ഇത് ലോകത്തിന്റെ പ്രധാന പ്രിന്റിംഗ് പ്രസ്സായ അമേരിക്കയിൽ നിന്നുള്ള കാന്റിലോൺ ഇഫക്റ്റിന്റെ ആരംഭമാണ്.
യുദ്ധ വ്യവസായത്തിലൂടെയുള്ള യുദ്ധത്തിന്റെ മറവിൽ സമ്പദ്‌വ്യവസ്ഥയെ ഉത്തേജിപ്പിക്കുകയാണിത്. ഓരോ തവണയും പണം അച്ചടിക്കുമ്പോൾ, അത് ലോകമെമ്പാടുമുള്ള എല്ലാവരുടെയും വാങ്ങൽ ശേഷി കവർന്നെടുക്കുകയാണ് അല്ലെങ്കിൽ പണത്തിന്റെ മൂല്യം കവർന്നെടുക്കുകയാണ്. എല്ലാ രാജ്യങ്ങളെയും കൊള്ളയടിക്കുന്നു. ഡോളറിൽ ഉയർന്ന കരുതൽ ശേഖരമുള്ള രാജ്യങ്ങളെ കൂടുതൽ കൊള്ളയടിക്കുന്നു. യുദ്ധത്തിൽ തോറ്റാലും ജയിച്ചാലും അവർ കാര്യമാക്കുന്നില്ല. അവർക്ക് ലോകത്തെ എത്രത്തോളം കൊള്ളയടിക്കാമെന്ന് മാത്രമേ അവർ ശ്രദ്ധിക്കുന്നുള്ളൂ.
