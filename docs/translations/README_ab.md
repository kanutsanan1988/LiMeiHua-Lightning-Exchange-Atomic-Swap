Иара аибашьра ду аҟаҵара ахьырхәашара мацара иҟаиҵоит
Иара иҟаиҵоит аибашьра аҟынтә апату змамзаара
Избанзар аибашьра аџьара ауп, аџьарагьы аибашьра ауп. Аҭоурых ҳазҳәоит убас. Апату амамзар, аибашьра аҟаҵаразы агәахәарагьы ыҟам.

Аибашьра аԥара аҭыжьразы ауп
Афинансқәа рминистрра аибашьратә облигациақәа ҭнажьуеит, аџьынџьуаа аанырхәырц азы ааԥхьара ҟанаҵоит. Знык иаанырхәомзар, ацентртә банк аанырхәоит. Уаҳа аԥара аҭыжьра ишалагалоу.
